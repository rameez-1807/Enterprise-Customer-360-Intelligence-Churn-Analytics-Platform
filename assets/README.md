# Enterprise Customer 360 Intelligence & Churn Analytics Platform
## Static Assets Directory

This directory contains static assets used across the platform:

- **icons/** — SVG and PNG icon assets for dashboard navigation
- **images/** — Brand logos, background images, and visual assets
- **fonts/** — Custom typography files (if not sourced via CDN)

### Asset Guidelines

- Use SVG format for icons and logos where possible for scalability.
- Compress PNG/JPEG images to optimize dashboard load performance.
- Follow the enterprise naming convention: `asset_type_description.ext`
