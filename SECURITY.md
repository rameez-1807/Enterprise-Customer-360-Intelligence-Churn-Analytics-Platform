# Security Policy

## Supported Versions

Only the latest production releases are supported with active security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this enterprise platform seriously. If you discover a vulnerability, please do NOT disclose it publicly in issues or pull requests.

Instead, please send an email reporting the vulnerability to [security@customer360.com](mailto:security@customer360.com).

Please include:
* A detailed description of the vulnerability.
* Steps to reproduce the exploit (proof of concept code or scripts).
* Potential business or database impact.

We will review your submission and respond within **48 hours** with a plan for remediation.
